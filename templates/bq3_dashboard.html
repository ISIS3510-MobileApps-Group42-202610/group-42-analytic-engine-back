Methodological Definitions

  • Session started: first search_started or filter_applied event in a session.
  • Meaningful interaction: listing_opened, message_sent, or reservation_created.
  • Interaction rate: percentage of started sessions that reached at least one meaningful interaction.
  • Elapsed time: difference between first search/filter event and first meaningful interaction in the same session.
